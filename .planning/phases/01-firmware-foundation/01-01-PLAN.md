
`_calc_checksum()` — Rotating-accumulate algorithm (NOT CRC32). Copy exactly from pedal_config.c:
```c
static uint32_t _calc_checksum(const retro_config_t *config) {
    const uint8_t *data = (const uint8_t *)config;
    size_t len = offsetof(retro_config_t, checksum);
    uint32_t sum = 0;
    for (size_t i = 0; i < len; i++) {
        sum += data[i];
        sum = (sum << 3) | (sum >> 29);
    }
    return sum ^ 0xDEADBEEF;
}
```

`config_update_checksum()` — Sets config->checksum = _calc_checksum(config).

`config_is_valid()` — Returns true if magic == CONFIG_MAGIC AND version == CONFIG_VERSION AND checksum == _calc_checksum(config).

`config_load()` — Reads from FLASH_CONFIG_ADDR via memcpy. If config_is_valid fails, calls config_set_defaults.

`config_save()` — Allocates uint8_t buf[FLASH_PAGE_SIZE], memset to 0xFF, memcpy config into it, then save_and_disable_interrupts(), flash_range_erase(FLASH_CONFIG_OFFSET, FLASH_SECTOR_SIZE), flash_range_program(FLASH_CONFIG_OFFSET, buf, FLASH_PAGE_SIZE), restore_interrupts().

`config_set_defaults()` — Sets:
- magic = CONFIG_MAGIC
- version = CONFIG_VERSION
- All pin_buttons[i] = -1 (disabled)
- pin_lt = -1, pin_rt = -1
- mode_lt = INPUT_MODE_DIGITAL, mode_rt = INPUT_MODE_DIGITAL
- debounce_ms = 5
- device_name = "" (memset to 0)
- lt_min = 0, lt_max = 4095, rt_min = 0, rt_max = 4095
- lt_invert = 0, rt_invert = 0
- lt_ema_alpha = 0, rt_ema_alpha = 0 (0 = fastest response, matching guitar convention)
- Calls config_update_checksum at the end
